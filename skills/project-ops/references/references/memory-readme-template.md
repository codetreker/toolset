# 项目 Memory README 模板

位置：`memory/projects/{project-name}/README.md`

README.md 是项目的精华汇总——概述 + 重要决策 + 里程碑 + 设计变更 + 经验教训。
详细日志记到 daily notes。

```markdown
# {项目名}

- **频道**：#{频道名}（`频道ID`）
- **仓库**：https://github.com/{org}/{repo}
- **技术栈**：{主要语言/框架}
- **启动时间**：YYYY-MM-DD
- **状态**：active

## 项目描述

（一段话说清楚这个项目是干什么的）

## 团队

- 总管/架构：飞马
- 开发：战马
- 产品：野马
- QA：烈马

## 重要决策

（随项目进展补充：决策内容 + 决策人 + 理由）

## 里程碑

（Phase/阶段完成记录）

## 设计变更

（架构调整、接口变更、方案推翻等）

## 经验教训

（踩坑、反直觉的行为、值得备忘的细节）
```
