# Feature Lifecycle (DEPRECATED)

> ⚠️ **此 skill 已废弃。请使用以下两个 skill 替代：**
>
> - **feature-developflow** — 开发流程：讨论 → 需求确认 → PRD → UI 线框图 → 技术设计 → 开发 → Review → QA test 环境验收 → PR 合并
> - **service-releaseflow** — 发布流程：Deploy staging → QA staging E2E 回归 → Approve → Deploy prod → Prod 验证
>
> 拆分原因：开发和发布是两个独立的关注点，触发场景不同，负责人侧重不同。合在一起容易跳步或混淆阶段。
>
> —— 2026-04-26 拆分
